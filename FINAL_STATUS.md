## 🚀 DEPLOYMENT

### **Local (Docker)**
```bash
docker-compose up
open http://localhost:3000
```

### **Production (Render)**
1. Go to https://dashboard.render.com
2. New → Blueprint
3. Select repo: `converto-business-os-quantum-mvp-1`
4. Add `OPENAI_API_KEY` environment variable
5. Deploy!

**Estimated Cost:** $7-37/month
